# 🚀 Lumo – Animated Gadget Website UI

🌐 **Live Demo:**
👉 [https://lumo-nbb9.onrender.com](https://lumo-nbb9.onrender.com)

---

## 🎨 Overview

Lumo is a **modern, design-focused gadget website UI** built using **HTML, CSS, JavaScript, and GSAP**.
The project focuses on **smooth animations, responsive design, and interactive UI**, without any backend functionality.

> ⚠️ This is a **frontend/UI project only** — no cart, authentication, or backend logic is included.

---

## ✨ Features

### 🎬 Animations

* GSAP loader animation with timeline
* Smooth page reveal transitions
* Scroll-based animations using ScrollTrigger

### 🖼️ Interactive UI

* Auto image slider (JavaScript)
* Infinite scrolling brand logos
* Infinite customer reviews carousel

### 📱 Responsive Design

* Mobile-first approach
* Works on:

  * Mobile (portrait & landscape)
  * Tablet
  * Desktop

### 🧭 Navigation

* Desktop navbar
* Slide-in mobile menu
* Smooth scroll to contact/footer section
* Multi-page UI (Home & About)

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3** (Flexbox, Grid, Media Queries)
* **JavaScript (Vanilla JS)**
* **GSAP (GreenSock Animation Platform)**
