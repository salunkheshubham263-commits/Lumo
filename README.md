Lumo – Animated Gadget Website UI

Lumo is a modern, design-focused gadget website UI built using HTML, CSS, JavaScript, and GSAP.
This project focuses on visual design, smooth animations, and responsive layout, not backend functionality.

🌐 Overview
Lumo is a front-end concept website created for gadget lovers, showcasing products and services through an engaging and interactive interface.

⚠️ This is a UI/UX project only — no backend, cart, or real product functionality is included.

✨ Features
🎬 Smooth Animations
Loader animation using GSAP timeline

Page reveal transitions

Scroll-based effects using ScrollTrigger

🖼️ Interactive Design
Auto image slider in hero section

Infinite scrolling brand logos

Infinite customer reviews carousel

📱 Responsive Layout
Fully responsive design

Optimized for:

Mobile (portrait & landscape)

Tablet

Desktop

🧭 Navigation UI
Desktop navbar + mobile menu
Slide-in mobile navigation panel
Smooth scroll to footer contact section
Multi-page UI (Home & About pages)

🛠️ Tech Stack
HTML5
CSS3
Flexbox
Grid
Media Queries
JavaScript (Vanilla JS)
GSAP (GreenSock Animation Platform)
